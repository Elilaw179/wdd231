Project Site Plan

BYU-Idaho WDD 131

Site Name

Adventure Travel Blog

This website is a travel blog dedicated to adventure seekers. It provides travel guides, hiking tips, and personal experiences from around the world.

Site Purpose

The website will serve as a resource for adventure travelers, offering travel tips, packing guides, and destination recommendations.

Scenarios

  • What are the top 10 best hiking trails in North America?
  • How can I prepare for a solo backpacking trip?

Color Schema

Primary Color: #1E3A5F (Navy Blue) - Used for headings and accents

Secondary Color: #F4A261 (Orange) - Used for buttons and highlights

Typography

Heading Font: Montserrat (Bold, 700) - Used for all headings

Body Font: Open Sans (Regular, 400) - Used for paragraph text
